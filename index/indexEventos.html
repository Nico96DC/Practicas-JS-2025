Ejemplos de 15 Eventos en JavaScript

Descripción de los eventos:

  1. click: Se activa al hacer clic en un elemento.
  2. dblclick: Se activa al hacer doble clic en un elemento.
  3. mouseover: Se activa cuando el puntero del mouse pasa sobre un elemento.
  4. keydown: Se activa cuando se presiona una tecla mientras un elemento tiene el foco.
  5. keyup: Se activa cuando se suelta una tecla mientras un elemento tiene el foco.
  6. input: Se activa cuando el valor de un campo de entrada cambia.
  7. change: Se activa cuando el valor de un campo de formulario cambia y pierde el foco.
  8. submit: Se activa cuando se envía un formulario.
  9. load: Se activa cuando una página o imagen ha terminado de cargarse.
  10. scroll: Se activa cuando se desplaza la barra de desplazamiento de un elemento.
  11. resize: Se activa cuando se cambia el tamaño de la ventana del navegador.
  12. contextmenu: Se activa al hacer clic derecho en un elemento para abrir el menú contextual.
  13. focus: Se activa cuando un elemento recibe el foco.
  14. blur: Se activa cuando un elemento pierde el foco.
  15. drag & drop: Eventos relacionados con arrastrar y soltar elementos.
